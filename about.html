faq

when did you start drawing?

this is going to be the most cliche answer ever lol. I started doodling on random pieces of paper when I was a kid and I started drawing seriously when I was 16 or so!


did you go to art school?

I went to animation school for three years and I'm now working in UI/UX design. oh, how things change and how unexpected outcomes happen!


which tablet do you use/are you using [insert-tablet-name]?

I draw using a Wacom Intuos Pro Medium (PTH-660) tablet and Clip Studio Paint EX.


do you still draw traditionally?

I love drawing digitally and it has been my favourite medium for a long, long time now. I don't think I'll go back to primarily drawing using traditional media anytime soon! I still do draw traditionally from time to time.
